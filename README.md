# Prune Things

Tools for cleaning up your system and freeing up space. 

![](prune.jpg)

__Use at your own risk.__

## Command Line

### Docker

```
docker system prune --all
```

### Conda

```
conda clean --all
```

### Git

```
git gc --aggressive --prune=now
```

### Homebrew

```
brew cleanup -s
```

### Mac/Linux Cache Folder

```
rm -r ~/.cache
```
